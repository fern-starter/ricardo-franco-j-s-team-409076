**ES:**

- `GET /list-products` → catálogo de productos.
- `POST /initiate-payment` → iniciar pago.
- `POST /validate-transaction` → validar OTP.
- `GET /order-status/{order_id}` → estado del pedido.
- `POST /refund/{order_id}` → procesar reembolso.

**EN:**

- `GET /list-products` → product catalog.
- `POST /initiate-payment` → initiate payment.
- `POST /validate-transaction` → validate OTP.
- `GET /order-status/{order_id}` → order status.
- `POST /refund/{order_id}` → process refund.
