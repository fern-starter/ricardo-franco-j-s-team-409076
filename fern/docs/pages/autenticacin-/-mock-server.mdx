**ES:**

1. Crear la colección con endpoints y ejemplos.
2. Añadir **Examples** en cada request.
3. Seleccionar colección → botón **Mock Collection**.
4. Postman genera una URL pública.
5. Probar la URL: devuelve las respuestas simuladas.

**EN:**

1. Create the collection with endpoints and examples.
2. Add **Examples** in each request.
3. Select collection → click **Mock Collection**.
4. Postman generates a public URL.
5. Test the URL: it returns the simulated responses.
