title: Overview
---

**ES:** Contiene los endpoints para iniciar y procesar pagos. Usa estas solicitudes para comenzar un nuevo flujo de pago, enviar los datos de pago y recibir confirmación de una transacción exitosa o fallida.

**EN:** Contains endpoints for initiating and processing payments. Use these requests to start a new payment flow, submit payment details, and receive confirmation of a successful or failed transaction.
