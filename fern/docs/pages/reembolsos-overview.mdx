---
title: Overview
---

**ES:** Contiene los endpoints para procesar reembolsos sobre pedidos existentes. Usa estas solicitudes para activar un reembolso dado el ID de un pedido, revirtiendo un pago previamente completado y devolviendo los fondos al cliente.

**EN:** Contains endpoints for processing refunds on existing orders. Use these requests to trigger a refund for a given order ID, reversing a previously completed payment and returning funds to the customer.
