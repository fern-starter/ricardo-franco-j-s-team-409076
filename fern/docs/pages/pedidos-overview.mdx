title: Overview
---

**ES:** Contiene los endpoints para consultar información de pedidos. Usa estas solicitudes para obtener el estado actual de un pedido específico mediante su ID, permitiendo rastrear su progreso a lo largo del ciclo de cumplimiento.

**EN:** Contains endpoints for querying order information. Use these requests to retrieve the current status of a specific order by its ID, allowing you to track progress through the fulfillment lifecycle.
